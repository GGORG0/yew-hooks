# Yew Hooks

Hooks for [Yew](https://github.com/yewstack/yew), inspired by [streamich/react-use](https://github.com/streamich/react-use) and [alibaba/hooks](https://github.com/alibaba/hooks).

## Hooks

- use_toggle / use_bool_toggle.
- use_counter.
